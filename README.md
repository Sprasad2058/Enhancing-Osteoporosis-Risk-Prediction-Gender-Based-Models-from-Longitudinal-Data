# Enhancing-Osteoporosis-Risk-Prediction-Gender-Based-Models-from-Longitudinal-Data

This project focuses on improving osteoporosis risk prediction through machine learning models trained separately for males and females using longitudinal health data. The study integrates clinical and biochemical factors including age, diabetes status, hormone levels, and bone density indicators to enhance early detection of hip osteoporosis and potential fracture risk.
